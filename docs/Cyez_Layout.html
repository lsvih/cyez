Class: Layout

Cyez#Layout(layout)

new Layout(layout)

将布局算法应用与全部节点。不能在一次布局完成前进行第二次布局。
Parameters:
Name Type Description
layout String 指定一种布局方式，可选值有：['grid', 'circle', 'concentric', 'breadthfirst', 'cose', 'cola', 'dagre', 'elk', 'klay', 'spread']
